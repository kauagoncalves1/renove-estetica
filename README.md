# 💆 Renove Estética Avançada

Site institucional com chatbot de IA integrado para uma clínica de estética fictícia, desenvolvido como projeto de estudo full-stack.

---

## ✨ Funcionalidades

- **Landing page completa** — Hero, Sobre, Serviços, Depoimentos e Contato
- **Chatbot com IA** — Widget flutuante com respostas em tempo real sobre a clínica
- **Tema claro/escuro** — Toggle com persistência no localStorage
- **Animações de scroll** — Elementos animados via Intersection Observer API
- **Imagens profissionais** — Integração com Unsplash
- **Design responsivo** — Adaptado para desktop e mobile

---

## 🤖 Chatbot

O chatbot utiliza o modelo **LLaMA 3.3 70B** via **Groq API** (gratuita) e responde perguntas sobre serviços, preços, horários, localização e agendamentos.

---

## 🛠️ Stack

| Tecnologia | Uso |
|---|---|
| Next.js 14 (App Router) | Framework principal |
| TypeScript | Tipagem estática |
| Tailwind CSS v4 | Estilização |
| Groq SDK (LLaMA 3.3) | IA do chatbot |
| Vercel | Deploy |

---

## 📱 WhatsApp Bot

A integração com WhatsApp via **Evolution API** (Docker) foi estruturada e documentada, porém não foi ativada por questões de privacidade — o bot requer vinculação a um número de WhatsApp real, e optei por não expor meu número pessoal em um projeto de estudo público.
